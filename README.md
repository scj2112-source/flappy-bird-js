# Flappy Bird JS

Flappy Bird JS는 JavaScript, HTML, CSS로 구현된 간단한 플래피 버드(Flappy Bird) 클론 게임입니다.  
플레이어는 스페이스바 또는 마우스 클릭으로 새를 조작하여 파이프 사이를 통과하는 것이 목표입니다.  
이 프로젝트는 웹 브라우저에서 동작하며, 누구나 쉽게 실행하고 커스터마이즈할 수 있습니다.

## 게임 설명

- 사용자는 새(Flappy Bird)를 조작하여 장애물(파이프) 사이를 통과해야 합니다.
- 새는 중력에 의해 아래로 떨어지며, 스페이스바 또는 마우스 클릭으로 위로 날 수 있습니다.
- 파이프에 부딪히거나 화면 아래로 떨어지면 게임이 종료됩니다.
- 통과한 파이프의 개수에 따라 점수가 올라갑니다.

## 코드 구조

- `index.html` : 게임 실행을 위한 HTML 파일입니다. 캔버스(canvas) 요소와 스크립트가 포함되어 있습니다.
- `style.css` : 게임의 기본 스타일을 담당합니다. 캔버스의 위치와 배경 등을 지정합니다.
- `main.js` (또는 `script.js`) : 게임의 핵심 로직이 구현되어 있습니다.
  - 새(Bird) 객체 생성 및 움직임 처리
  - 파이프(Pipe) 생성 및 이동 처리
  - 충돌 감지 및 점수 계산
  - 게임 루프 및 애니메이션 처리
  - 사용자 입력(키보드, 마우스) 이벤트 처리

## 실행 방법

1. 저장소를 클론합니다.
   ```bash
   git clone https://github.com/scj2112-source/flappy-bird-js.git
   ```
2. 프로젝트 폴더로 이동합니다.
   ```bash
   cd flappy-bird-js
   ```
3. `index.html` 파일을 웹 브라우저에서 엽니다.

## 커스터마이즈

- 새, 파이프, 배경 등의 이미지를 변경하거나, 난이도(파이프 간격, 속도 등)를 조절할 수 있습니다.
- 코드의 주석을 참고하여 원하는 기능을 추가하거나 수정해보세요.
